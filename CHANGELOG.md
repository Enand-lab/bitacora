# Changelog

Todos los cambios notables en Cuaderno de Bitácora se documentarán aquí.

El formato sigue [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.1.0] - 2025-11-27

### Añadido
- Primera versión pública del Cuaderno de Bitácora.
- Diario offline-first con soporte para notas, metadatos y fotos.
- Integración opcional con Signal K (posición, estado, sincronización de notas).
- API REST para automatización (Node-RED, scripts).
- Soporte multilingüe: español, inglés, francés, chino, ruso y japonés.
- Copia de seguridad manual vía UI o API.
- Instalación automatizada mediante `install.sh` (Raspberry Pi / Linux).
- PWA (Progressive Web App) para uso en móviles.
